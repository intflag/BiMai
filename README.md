### 项目描述
* BiMai是一个线上商城，分为前台和后台，具有订单管理、公告管理、商品管理、用户管理等功能。
### 技术栈
* Servlet，JSP，MySQL，JavaMail发送邮件，易宝支付SDK，文件上传下载，C3P0连接池，DBUtils等技术；
### 开发环境
* MyEclipse10，JDK1.7，Tomcat7.0.57，MySQL5.7；
